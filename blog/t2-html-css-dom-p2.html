Tech Talk

What is grid based design?

This is a format which most websites follow. It allows objects and/or elements to be placed on a grid and thus are easier to edit, arrange, and configure.

What is all the hype about responsive webpages?

This is a sort-of organizational technique. The "hype" arises because we are increasingly viewing webpages from a variety of devices, such as mobile phones and tablets. A successfully responsive web design will reflect ideally as per the device accessing it.

What is semantic structure?

Semantic structure is structure that is clearly defined to both man and machine. E.g. table, color, and body.
